Codedoc for Pi project - strand of lights

python scripts and what they do:
udp_prot.py - just reads in data from touchdesign in hexadecimal format - from lenovo
plosne_disko.py - lights up all LEDs in strand with same colour, given by a randomly selected integer
randno_disko.py - lights up each individual LED with random colour (also randint)
data_disko.py - lights up LEDs according to udp-received data from TD
